# wick

[![Python 3.6](https://img.shields.io/badge/python-3.6-blue.svg)]()

wick is a Python command line utility for generating file IO code from C structs.

## Installation

```shell
$ pip install wick
```

## Usage

```shell
$ wick common.h --language=python
```
